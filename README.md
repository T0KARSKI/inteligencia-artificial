# Estudo Dirigido 1 - Inteligência Artificial

Implementação do problema Lights Out utilizando diferentes algoritmos de busca:

- Bsuca em Largura (bfs.py)
- Busca em Profundidade (dfs.py)
- Busca Gulosa (greedy.py)
- Busca Local (hill_climbing.py)
- Algoritmo A (astar.py)
- utils.py
- main.py
- .gitignore

Instruções: 

-> Faça o Download.ZIP do repositório

-> Abra um terminal CMD / Powershell na pasta inteligencia-artificial-main\codigo-fonte

-> bash: python main.py
